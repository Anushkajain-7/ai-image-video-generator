# 🎨 AI Image & Video Generator

A deep learning project that generates images and short videos from text prompts
using Stable Diffusion and Gradio, deployed on Google Colab.

## 🚀 Features
- Text-to-Image generation using Stable Diffusion v1.5
- Text-to-Video generation (frame-by-frame)
- Interactive Gradio web UI
- Public shareable link via Colab

## 🛠️ Tech Stack
- Stable Diffusion v1.5 (runwayml)
- Hugging Face Diffusers
- Gradio
- PyTorch
- Google Colab (T4 GPU)

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Change runtime to GPU (T4)
3. Add your Hugging Face token
4. Run all cells in order
5. Open the Gradio public URL
